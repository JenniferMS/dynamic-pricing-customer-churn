# Dataset Information

This project uses the Dunnhumby – The Complete Journey dataset from Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/frtgnn/dunnhumby-the-complete-journey

The full raw dataset is not included in this repository due to size and licensing considerations.

To reproduce the project:
1. Download the dataset from Kaggle
2. Place the CSV files into the local data directory
3. Update paths if necessary
